Interpretation Of Requirement

For this assignment, our skills in Web Development was tested. The following tasks were required:

  1. Part 1: Create a XML version of our Resumes and be able to display it in a webpage
  2. Part 2: Provide a multi-page tutorial website with interactive quizzes
  3. Part 3: Display a photo slideshow that is manageable with JSON
  4. Part 4: Create a webpage that has multiple utilities tools suite

My Implementation

  1. Part 1: I started out simply transforming my resume into an XML and then writing the xsl for it to be shown in a webbrowser. After that I set up my TMA1 web pages and ensured it was accessible.
  2. Part 2: This part took most of my time, I had setup the general outline of each unit and their quizzes with HTML5 and CSS. I then worked on the quiz system which includes XML, Javascript, and AJAX. After this I populated the tutorials and quizzes.
  3. Part 3: I collected photos from my co-op term in Burnaby, BC. I create the JSON array and made a simple slide show. I then developed the play and stop and next/prev buttons. After that I spend time on the effects! This was done with HTML5 and Javascript
  4. Part 4: This was quite straightforward using input HTML5 elements and Javascript, where the calculations are. For the values of conversion for weight, I used the one on google.ca. The mortgage calculator was based off of this website. Lastly I thought that a tip calulator would be extremely useful. It simply allows users to be able to calculate their total bill based on tip percentage or tip amount.
