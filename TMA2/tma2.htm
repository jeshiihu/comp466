Part 1: Bookmarking Service


For this part of the assignment, we were to create an online bookmarking service to store a list of URLs. It should allow users to be able to add, edit, and delete any websites. While doing so the service will always check if the URL entered is a valid website.


Before logging in, you will be shown the Top 10 list of the most popular URLs that other users have added to their lists. You can begin by logging into an existing account or signing up as a new user. Once logged in, you can modify your url lists!


This part was done using html, css, javascript/jquery, and php. I first created the various pages and then worked through the homepage followed by the individual user pages. This part accesses the MySQL database that is set up on the server.


There are two tables:

  1. Users: Username (Primary Key), Password
  2. Bookmarks: BookmarkID (Primary Key), Url, User (Foreign Key)


This part of the assignment was created using a lot of dynamic DOM manipulations to transform the visual GUI of the website while asynchronously posting and getting php requests to the server. It also makes use of cookies in order to keep the user logged in.
